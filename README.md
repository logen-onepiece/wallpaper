## 💡 使用技巧

### 批量删除
1. 点击右上角"批量删除"按钮
2. 单击壁纸选中（可多选）
3. 点击"全选"快速选中所有
4. 点击"删除选中"批量删除
5. 点击"取消"退出批量模式

### 显示模式切换
- **📐 自适应**：保持原始比例，不裁切
- **🖼️ 拉伸填充**：填满容器，可能裁切
- **⬛ 完全拉伸**：强制填满，可能变形

### 全屏查看
- **进入全屏**：点击壁纸
- **切换壁纸**：方向键 ← → 或触摸滑动
- **退出全屏**：点击任意位置或按 ESC

## 📞 常见问题

**Q: 为什么首次访问显示"加载中"？**
A: IndexedDB 初始化需要时间，1-2 秒后会显示真实容量。

**Q: 数据会丢失吗？**
A: 不会！IndexedDB 数据永久存储在浏览器中，除非手动清除浏览器数据。

**Q: 不同浏览器数据会同步吗？**
A: 不会。每个浏览器独立存储。

**Q: 可以导出数据吗？**
A: 可以在浏览器控制台手动导出（参见 UPGRADE_GUIDE.md）。

**Q: 本地版本还保留吗？**
A: 是的！本地版本（index.html）仅在本地使用，不会部署到线上。
