# ⚡ 快速开始指南

## 🎯 5 分钟上手

### 1. 打开网站
访问：**http://localhost:8000** （或你部署的线上地址）

### 2. 上传第一张壁纸
1. 点击右上角 **"📤 上传壁纸"** 按钮
2. 选择一张图片或视频
3. 等待提示"✅ 成功上传 1 个文件！"

### 3. 查看壁纸
- 图片会自动出现在网格中
- 视频会自动播放预览

### 4. 调整显示模式
1. **鼠标悬停**在壁纸上
2. 点击**左上角图标**（📐）
3. 循环切换：自适应 → 拉伸 → 完全拉伸

### 5. 全屏欣赏
1. **点击壁纸**进入全屏
2. **左上角**显示当前时间
3. 用**方向键**或**滑动**切换下一张
4. **点击屏幕**或按 **ESC** 退出

## 🎨 显示模式选择

### 📐 自适应填充（默认）
**适合**: 想看到完整画面
- ✅ 完整显示所有内容
- ❌ 可能有黑边
- 推荐：风景照、艺术图

**适合**: 大多数壁纸
- ✅ 填满整个屏幕
- ✅ 保持比例
- ⚠️ 可能裁剪边缘
- 推荐：电影画面、游戏截图

### ⬛ 完全拉伸
**适合**: 特殊尺寸
- ✅ 强制填满
- ❌ 可能变形
- 推荐：特殊比例图片

## 💾 存储管理

### 容量限制
- **Chrome/Edge**: 10 MB
- **Safari**: 5 MB
- 每个设备独立存储

### 建议配置
| 内容 | 单个大小 | 数量 | 总计 |
|------|----------|------|------|
| 图片 | 300KB | 20 张 | 6MB |
| 视频 | 2MB | 2 个 | 4MB |
| **合计** | - | - | **10MB** |

### 优化方法
**图片压缩**:
1. 访问 https://tinypng.com/
2. 上传图片
3. 下载压缩后的文件（节省 70%）

**视频压缩**:
```bash
# 压缩到 2MB 左右
ffmpeg -i input.mp4 -vf scale=1920:1080 -crf 28 output.mp4
```

## ⚠️ 常见问题

### Q1: 为什么刷新后壁纸还在？
A: 数据存储在浏览器 LocalStorage，不会丢失。

### Q2: 为什么其他设备看不到我的壁纸？
A: 数据存储在本地浏览器，不同设备独立存储。

### Q3: 如何备份数据？
A:
- 方法 1: 使用 Chrome 同步（登录 Google 账号）
- 方法 2: 保留原始文件
- 方法 3: 等待未来的导出功能

### Q4: 提示"存储空间已满"怎么办？
A:
1. 删除一些不需要的壁纸
2. 上传前压缩图片/视频
3. 使用更小的文件

### Q5: 动态壁纸无法播放？
A: 确保视频格式为 MP4 (H.264) 或 WEBM，这些格式浏览器原生支持。

### Q6: 显示模式怎么切换？
A: 鼠标悬停在壁纸上，点击**左上角**的图标按钮。

## 🚀 部署到线上

### Vercel（推荐）
```bash
# 1. 安装 CLI
npm install -g vercel

# 2. 部署
cd /Users/mac137/wallpaper-gallery
vercel

# 3. 获得线上地址
# 例如: https://my-wallpaper.vercel.app
```

### Netlify（拖拽）
1. 访问 https://app.netlify.com/drop
2. 拖拽整个文件夹
3. 等待 30 秒
4. 获得线上地址

### GitHub Pages
```bash
# 1. 创建仓库
git init
git add .
git commit -m "我的壁纸库"

# 2. 推送到 GitHub
git remote add origin https://github.com/用户名/仓库名.git
git push -u origin main

# 3. 启用 Pages
# 在仓库设置中启用 GitHub Pages
# 访问: https://用户名.github.io/仓库名
```

## 📊 使用统计

### 查看存储使用量
打开浏览器控制台（F12），运行：
```javascript
const data1 = localStorage.getItem('static_wallpapers') || '';
const data2 = localStorage.getItem('dynamic_wallpapers') || '';
const totalBytes = new Blob([data1, data2]).size;
const totalMB = (totalBytes / 1024 / 1024).toFixed(2);
const staticCount = JSON.parse(data1 || '[]').length;
const dynamicCount = JSON.parse(data2 || '[]').length;

console.log(`
📊 存储使用情况
━━━━━━━━━━━━━━━━━━
静态壁纸: ${staticCount} 张
动态壁纸: ${dynamicCount} 个

剩余空间: ${(10 - totalMB).toFixed(2)} MB
━━━━━━━━━━━━━━━━━━
`);
```
